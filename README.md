## 🌍 What is InstantSite AI?

**InstantSite AI** is an AI-powered website generator that converts simple text prompts into complete websites with modern UI, responsive design, and clean code.

Just type:

> “Modern portfolio website for a UI/UX designer”

And get:

- Full HTML/CSS/JS website
- Live preview
- Downloadable ZIP
- Deploy-ready project

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/yourusername/instantsite-ai.git

# Install dependencies
npm install

# Start development server
npm run dev
```

### 🔑 Environment Variables

Create `.env` file:

```env
GEMINI_API_KEY=your_gemini_api_key_here
```
